# Actions secrets

```
# Path to production environment files
PRODUCTION_DEPLOY_PATH=<path>
```

```
# Production environment variables
PRODUCTION_ENV_FILE=<base64 encoded .env file>
```

```
# Production SSH private key
PRODUCTION_SSH_KEY=<privkey>
```

```
# Production account username in server
PRODUCTION_SSH_USER=<username>
```

```
# Private container registry username
REGISTRY_UNAME=<username>
```

```
# Private container registry password
REGISTRY_PWORD=<password>
```

```
# CI/CD Robottest docker image
ROBOTTEST_IMAGE=<image name>
```

```
# Server address
SERVER=<address>
```

```
# Server's SSH port
SSH_PORT=<port>
```

```
# Path to staging environment files
STAGING_DEPLOY_PATH=<path>
```

```
# Staging environment variables
STAGING_ENV_FILE=<base64 encoded .env file>
```

```
# Staging SSH private key
STAGING_SSH_KEY=<privkey>
```

```
# Staging account username in server
STAGING_SSH_USER=<username>
```

```
# CI pipeline tests .env file
TESTS_ENV_FILE=<base64 encoded .env file>
```
